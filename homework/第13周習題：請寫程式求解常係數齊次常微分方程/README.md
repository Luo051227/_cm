## 常係數齊次線性常微分方程（Constant Coefficient Homogeneous Linear ODE）
標準形式（以二階為例）如下： $$ay'' + by' + cy = 0$$  

 $a, b, c$ 是常數， $y$ 是 $x$ 的函數。所謂「齊次」是指等號右邊為 $0$（沒有外力項）  

想像一個掛著重物的彈簧：  
* $y(t)$ (位置)：物體現在在哪裡
* $y'(t)$ (速度)：物體跑得有多快
* $y''(t)$ (加速度)：物體受到的力（牛頓第二定律 $F=ma$）

- $a$ (質量)：代表慣性，不想動或不想停
- $b$ (阻尼/摩擦力)：代表空氣阻力或油壓，它會讓物體慢下來（跟速度 $y'$ 有關）
- $c$ (彈簧硬度)：彈簧想把物體拉回原點的力量（跟位置 $y$ 有關）
- $= 0$ (齊次)：意思是 **「沒有人推它」**。你把物體拉開後放手，讓它自己動，這叫「自然反應」  

**為什麼要用 $e^{rx}$ 來猜答案？**  
$a \cdot (\text{某函數的兩次微分}) + b \cdot (\text{一次微分}) + c \cdot (\text{原函數}) = 0$  
$y$、 $y'$ 和 $y''$ 必須長得很像，才能互相加減抵銷變成 0  
世界上只有一種函數，不管微分幾次，長相都跟原本一樣，那就是指數函數 $e^{rx}$  
* $y = e^{rx}$
* $y' = r e^{rx}$
* $y'' = r^2 e^{rx}$

都有 $e^{rx}$，所以可以直接消掉，剩下的就是簡單的代數： $ar^2 + br + c = 0$


**核心觀念：特徵方程式 (Characteristic Equation)**
指數函數 $y = e^{rx}$ 是解答的原型
指因為指數函數的微分還是指數函數（ $y' = re^{rx}$, $y'' = r^2 e^{rx}$）

將 $y = e^{rx}$ 代入方程 $ay'' + by' + cy = 0$，我們可以消去 $e^{rx}$（因為 $e^{rx} \neq 0$），得到一個代數方程式，稱為特徵方程式： $$ar^2 + br + c = 0$$  

**三種解的情況 (Three Cases)**  
 根據特徵方程式的判別式 $\Delta = b^2 - 4ac$，根 $r$ 會出現三種情況，對應三種通解形式：  

| 判別式情況 | 根的形式 ($r_1, r_2$) | 通解 $y(x)$ | 說明 |
| :--- | :--- | :--- | :--- |
| **1. 相異實根**<br>($\Delta > 0$) | 兩個不同的實數<br>$r_1 \neq r_2$ | $$y = c_1 e^{r_1 x} + c_2 e^{r_2 x}$$ | 最單純的情況，直接線性疊加。 |
| **2. 重根**<br>($\Delta = 0$) | 兩個相同的實數<br>$r_1 = r_2 = r$ | $$y = c_1 e^{rx} + c_2 x e^{rx}$$ | 因為只有一個根，缺少第二個線性獨立解，所以必須乘上 $x$ 來補足。 |
| **3. 共軛複數根**<br>($\Delta < 0$) | $r = \alpha \pm i\beta$<br>($\alpha$是實部, $\beta$是虛部) | $$y = e^{\alpha x}(c_1 \cos \beta x + c_2 \sin \beta x)$$ | 利用歐拉公式將複數指數轉換為三角函數，解會出現震盪。 |

**高階方程式 (Higher Order ODEs)**
* 寫出特徵方程式（如 $r^3 - 1 = 0$）
* 解出所有的根
* 重根規則推廣： 如果某個根 $r$ 重複出現 $k$ 次，則對應的解為： $$e^{rx}, \quad xe^{rx}, \quad x^2 e^{rx}, \quad \dots, \quad x^{k-1}e^{rx}$$

**三種情況的「物理畫面」**  
* 情況一：相異實根 ($\Delta > 0$0 —— 「過阻尼」(Overdamped)
  * 數學：有兩個負的實數根，解答全是 $e^{-ax}$ 這種形式
  * 直覺：想像把彈簧丟進一桶濃稠的蜂蜜裡
  * 動作：把物體拉開，放手。因為摩擦力（阻尼 $b$）太強了，物體根本彈不起來，只會慢吞吞地滑回原點停下來
  * 特徵：不會震盪，直接歸零
* 情況二：共軛複數根 ($\Delta < 0$0 —— 「欠阻尼」(Underdamped)
  * 數學：根裡面有虛數 $i$。歐拉公式告訴我們 $e^{ix}$ 跟 $\sin, \cos$ 有關
  * 直覺：想像一個很有彈性的彈簧，在空氣中（摩擦力很小）
  * 動作：一放手，物體就會來回彈跳（震盪）。雖然因為有摩擦力，彈跳幅度會越來越小（ $e^{\alpha x}$ 負責變小），但它會一直晃
  * 特徵：會震盪  ($\sin, \cos$)，最後慢慢停下來
* 情況三：重根 ($\Delta = 0$) —— 「臨界阻尼」(Critically Damped)
  * 數學：兩個根一樣。解答需要補一個 $x$
  * 直覺：這是完美的平衡點。剛好在「會震盪」和「慢吞吞」之間
  * 動作：這是避震器設計最想要的狀態。不想車子一直晃（欠阻尼），也不想避震器硬到反應很慢（過阻尼）
  * 特徵：物體會以最快的速度回到原點，而且不會衝過頭（不震盪）

假設這三個系統的質量 ($a$) 和 彈簧硬度 ($c$) 都一樣，唯一的差別在於我們倒進去的油有多黏（阻尼 $b$ 不同）  
初始條件：  
* $y(0) = 1$：先把物體拉開 1 公尺
* $y'(0) = 0$：從靜止狀態放手（不給初速度）

1. 過阻尼 (Overdamped)：油太黏了
   方程式： $y'' + \mathbf{5}y' + 4y = 0$  (阻力係數 $b=5$，很大)  
* 特徵方程： $r^2 + 5r + 4 = 0 \Rightarrow (r+1)(r+4)=0$
* 根： $r = -1, -4$ (相異實根)
* 物理現象： 放手後，因阻力太大，彈簧像是被黏住一樣，慢慢地滑回原點。它完全不會超過原點（不會震盪）
* 數學解： $y(t) \approx 1.33e^{-t} - 0.33e^{-4t}$
 
2. 臨界阻尼 (Critically Damped)：剛剛好
   方程式： $y'' + \mathbf{4}y' + 4y = 0$ (阻力係數 $b=4$，調降了一點)
* 特徵方程： $r^2 + 4r + 4 = 0 \Rightarrow (r+2)^2=0$
* 根： $r = -2$ (重根)
* 物理現象： 這是最完美的狀態。放手後，它用最快的速度衝回原點，而且在碰到原點時剛好煞車停住，沒有衝過頭
* 數學解： $y(t) = e^{-2t} + 2te^{-2t}$  
 
3. 欠阻尼 (Underdamped)：太滑溜了
  方程式： $y'' + \mathbf{2}y' + 4y = 0$ (阻力係數 $b=2$，很小)
* 特徵方程： $r^2 + 2r + 4 = 0$
* 根： 公式解 $\frac{-2 \pm \sqrt{4-16}}{2} = -1 \pm \sqrt{3}i$ (共軛複數)
* 理現象： 阻力太小拉不住。放手後，它衝回原點時煞不住車，直接衝到負的方向，然後再彈回來，像溜溜球一樣來回震盪幾次才停下來
* 數學解： $y(t) = e^{-t}(\cos(\sqrt{3}t) + \frac{1}{\sqrt{3}}\sin(\sqrt{3}t))$
